Tips And Tricks

Status

For the moment, this project is still only an idea.

Description

The goal is to display tips that show information about    minecraft and allow players to learn more about the game.

There will be two ways to display information.

  1. The player will receive randomly a message in chat with the tips
  2. A popup will randomly appear on the player screen with the tips

        Server administrators can configure tips

  1. add new tips
  2. set the maximum number of time a specific tips will be display to a player
  3. set if a specific tips should be display in chat or with a popup
